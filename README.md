<details>
<summary><b>🤖 ML Stack</b></summary>
<br/>

| Component | Technology | Purpose |
|:---|:---|:---|
| Supervised Classifier | `CatBoost` + `XGBoost` + Logistic Regression | Threat probability per event |
| Anomaly Detector | `IsolationForest` | UEBA behavioral anomaly scoring |
| Text Features | `HashingVectorizer` (1–3 gram, 32K features) | Sparse NLP on raw log text |
| Explainability | `SHAP` | Per-event feature attribution for SOC analysts |
| Scaling | `StandardScaler` + `RobustScaler` | Feature normalization |
| Class Balancing | Custom upsampling | Handle imbalanced threat datasets |
| Risk Fusion | Weighted sigmoid formula | Combine ML + UEBA + TI signals → 0–100 score |

</details>
